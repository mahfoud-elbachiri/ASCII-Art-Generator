# ASCII Art Web Generator

A web-based ASCII art generator built using **Golang**.
Users input text, select a font style, generate ASCII output, and download it as `.txt`.

## Features
- Generate ASCII banners
- Fonts: standard, shadow, thinkertoy
- Download output as `.txt`
- Clean UI layout with gradients
- Cross-platform (Linux, macOS, Windows)

## Installation
```bash
git clone https://github.com/mahfoud-elbachiri/ASCII-Art-Generator.git
cd ascii-art-web
go mod tidy
go run .
```
## Access

Open browser:

http://localhost:8080


## Usage

1 - Enter text

2 - Select font

3 - Press Generate

Download .txt file
